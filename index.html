So bedienst du die Karte

Auf eine farbige Fläche tippen → Steckbrief der Gruppe (höchster Berg, Lage, Land). Rechts im Panel „Ebenen" schaltest du Färbung, Namen, Landesgrenzen, Gipfel, Hütten & Pässe; das Haus-Icon setzt die Standardansicht zurück.

Über diese Karte

Ein interaktiver Alpen-Atlas nach SOIUSA, der internationalen Gliederung der Alpen (italienischer Standard von 2005): alle 131 Gebirgsgruppen, gebündelt in fünf Großsektoren (Grandi Settori), dazu Gipfel, Hütten und Pässe aus OpenStreetMap.

SOIUSA auf Wikipedia →

Tech: Datenaufbereitung in Python, Rendering mit MapLibre GL JS, 3D-Terrain aus offenem DEM. Komplett statisch & keyless auf GitHub Pages (kein Server, kein API-Key). Quellcode auf GitHub →

Daten: SOIUSA (ARPA Piemonte) · OpenStreetMap – Gipfel/Hütten (ODbL) · Wikipedia/Wikidata – Steckbriefe (CC BY-SA) · Natural Earth · Esri World Imagery.
